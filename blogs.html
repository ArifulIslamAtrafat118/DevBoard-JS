What is event delegation in the DOM?

  • Event delegation uses a single event listener on a parent element to manage events from child elements.
  • It relies on event bubbling, where the event propagates upward through the DOM tree.
  • This approach improves performance and reduces memory usage.
  • It allows handling of dynamically added child elements without adding new listeners.

What is event bubbling in the DOM?

  • Event bubbling is the process where an event moves from the target element up through its ancestors.
  • The event is first handled by the child, then by parent, grandparent, etc.
  • This allows parent elements to respond to events from their children (used in event delegation).
  • The bubbling can be stopped using event.stopPropagation() in JavaScript.

How do you create, add, and remove elements using JavaScript?

  • Create: Use document.createElement("tag") to create a new element.
  • Add: Use parent.appendChild(child) or parent.prepend(child) to insert it into the DOM.
  • Remove: Use element.remove() to delete an element directly.
  • Alternative Remove: Use parent.removeChild(child) if you have the parent reference.
